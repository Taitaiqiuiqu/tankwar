# 坦克大战游戏

这是一个使用Python和Pygame模块开发的坦克大战游戏。游戏采用面向对象的设计思想，实现了经典坦克大战的基本功能。

## 功能特点

- 玩家控制英雄坦克进行游戏
- 敌人坦克具有AI行为
- 支持射击、爆炸特效
- 包含不同类型的墙壁和障碍物
- 背景音乐和音效

## 开发环境

- Python 3.x
- Pygame 模块

## 安装与运行

1. 确保已安装Python 3.x
2. 安装所需依赖：
   ```
   pip install pygame
   ```
   或者使用requirements.txt：
   ```
   pip install -r requirements.txt
   ```
3. 运行游戏：
   ```
   python main.py
   ```

## 游戏操作

- 方向键：控制坦克移动
- 空格键：发射子弹

## 文件结构

- `main.py`：游戏主程序入口
- `tank_war.py`：游戏主逻辑类
- `sprites.py`：游戏精灵类（坦克、子弹等）
- `settings.py`：游戏设置
- `resources/`：资源文件夹
  - `images/`：游戏图像资源
  - `musics/`：游戏音频资源
- `tools/`：工具脚本

## 游戏说明

- 控制你的坦克消灭敌人，保护基地
- 坦克可以发射子弹摧毁敌人和可破坏的墙壁
- 游戏中会出现各种障碍物，有些可以被摧毁，有些则不能
- 当敌人进入基地区域或者玩家坦克被摧毁时，游戏结束
